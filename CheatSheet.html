DOCTYPE
The DOCTYPE is needed, because the browser has to know which type of html is being used before he can render it
About the body element
The body element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
About the em element
The em is an phrase tag and it makes the text cursive.
About the head element
The head element is a storage room for head elements like (title, style, base, link, meta, script and noscript)
About the html element
The html tag tells the browser that it is an HTML document, it represents the root of an HTML document and it is a container for all HTML elements
About the strong element
The strong tag is also a phrase tag like the em element but this time it makes the text fat.
About the title element
The title element changes the little tab name on the top of the screen. It has to be in the head container. It also provides a name in case the website is added to the favorites, and a name is displayed when somebody searches it.

    Use combinators properly: direct child, descendant, adjacent sibling, sibling
  • descendant selector (space)
    •  div p{
    All the (paragraphs) which are between the < div >< /div > will be written in the same form as the class
    }
  • child selector (>)
    •  div > p{
    All the (paragraphs) which are between the < div >< /div > and have no other statements infront of them will be written in the same form as the class
    }
  • adjacent sibling selector (+)
    •  div + p{
    After < div >< /div > the directly following statement will printed the same way as the class
    }
    >
  • general sibling selector (~)
    •  div ~ p{
    All the (paragraphs) which are after the < div >< /div > and have no other statements infront of them will be written in the same form as the class
    }

    Block elements have which kind of characteristics?
    The CSS box model is essentially a box that wraps around every HTML element
    Inline elements have which kind of characteristics?
    Content - The content of the box, where text and images appear
    Padding - Clears an area around the content. The padding is transparent
    Border - A border that goes around the padding and content
    Margin - Clears an area outside the border. The margin is transparent
    What is the margin of an Html element? Where can one find it?
    Clears an area outside the border. The margin is transparent
    What is the padding of an Html element? How does it differ from margin?
    Clears an area around the content. The padding is transparent
